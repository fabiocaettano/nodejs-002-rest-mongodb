## NVM 

Instalar NVM no Ubuntu 20.04:

``` cli
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash 
source ~/.bashrc
nvm install node
```

Visualizar a vers찾o instalada:

``` cli 
nvm ls
```

Instalar outra vers찾o:

``` cli
nvm install 12.18.3
```

Checar as vers천es disponiveis:

``` cli
nvm ls-remote
```

Checar a vers찾o latest:

``` cli
nvm run --default --version
```
